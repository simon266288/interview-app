# v0.0.1 发布说明

## 版本信息
- 发布形态：单文件版本（index.html）
- 目标：快速可用、离线运行、完整题库与核心答题流程

## 主要特性
- 专项练习（8 模块，≥80 题）
- 选择/判断题即时反馈 + 解析
- 记忆卡片（翻转），“记住了/没记住”与错题本联动
- 错题本与收藏夹（列表进入刷题、支持删除）
- 模拟考试（随机 20 题，交卷后评分与答题卡）
- 首页统计（已刷题数/正确率/坚持天数占位）
- 本地持久化（localStorage，键：`interview_app_v1`）

## 已知问题与限制
- 模拟考试未实现倒计时组件
- 成绩页点击题号查看解析功能提示“开发中”
- 选择题暂不支持多选
- 无自动化测试与工程化构建

## 升级指引（面向 v0.0.2）
- v0.0.2 将采用工程化分层（Vite + Vue3 + TS + Pinia + Router + Tailwind）
- 保持 v0.0.1 的 UI 风格与交互体验
- 提供数据迁移（`interview_app_v1` -> `interview_app_v2`）
