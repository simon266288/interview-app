# v0.0.1 需求快照（单文件版）

## 概述
- 交付形态：单页 Web 应用，一个 `index.html` 包含全部代码（HTML/CSS/JS），本地直接打开即可运行
- 面向用户：梅军军（前端工程师）面试刷题
- 产品定位：驾考宝典式互动刷题，模块化练习 + 即时反馈 + 错题本 + 收藏夹 + 模拟考试

## 功能清单（已实现）
- 首页：六宫格入口（专项练习/模拟考试/错题本/收藏夹/记忆卡片）
- 模块练习：8 个模块（JS/CSS、Vue 原理、性能优化、工程化、低代码平台、浏览器与网络、手写代码、场景题）
- 题型支持：
  - 选择题（单选）与判断题：提交后即时显示对错与解析
  - 记忆卡片：点击翻转查看答案，支持“记住了/没记住”
- 错题本：自动收录答错题与“没记住”的卡片，支持从列表重做与删除
- 收藏夹：题目可收藏，收藏列表可直接进入刷题
- 模拟考试：随机抽取 20 题组卷，交卷后显示得分与答题卡概览（解析查看入口后续优化）
- 统计概要：首页显示“已刷题数/正确率/坚持天数（占位）”

## 数据与持久化
- 存储方案：`localStorage`
- 存储键：`interview_app_v1`
- 结构含义（v0.0.1）：
  - `completedIds: number[]`：已做对的题 ID
  - `wrongIds: number[]`：错题或“没记住”题 ID
  - `favoriteIds: number[]`：收藏题 ID
  - `moduleStats: Record<string, number>`：模块进度（占位）
  - `examHistory: []`：考试记录（预留）

## 题库
- 数量：≥ 80（各模块约 10 题）
- 类型覆盖：选择/判断/记忆卡片
- 特点：题目与解析紧贴简历中的技术栈与项目经验

## 使用方式
- 直接双击 `index.html` 或通过本地静态服务器打开
- 推荐预览路径：`http://localhost:8000/v0.0.1/index.html`

## 已知限制（v0.0.1）
- 模拟考试暂未实现倒计时组件（交卷后评分正常）
- 成绩页点击题号查看解析功能提示“开发中”
- 选择题暂不支持多选（v0.0.2 领域层将支持）
