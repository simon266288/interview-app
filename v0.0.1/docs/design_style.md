# v0.0.1 设计风格快照（单文件版）

## 风格关键词
- 干净、清晰、移动端优先
- 白底 + 蓝色主色调
- 大字号、大按钮、易触控区域

## 视觉规范
- 颜色：
  - 主色：蓝（按钮、选中态、强调）
  - 成功：绿（答对）
  - 错误：红（答错）
  - 背景：浅灰（页面背景）
  - 文本：深灰（题干/正文）、次要文字浅灰
- 字体与层级：
  - App 标题：约 20px，粗体
  - 题目文字：约 18px，行距舒适
  - 选项文字：约 16px
  - 解析文字：约 14px，浅灰说明文本

## 布局结构
- 首页：六宫格入口（图标 + 标签），卡片化，阴影轻微
- 答题页：
  - 顶部：返回、模块名/状态、题号进度
  - 中部：题干卡片 + 选项列表（纵向大按钮）
  - 反馈：选项立即变色（绿对红错），解析区浅蓝背景
  - 底部：上一题/下一题/收藏，固定底栏
- 记忆卡片：点击翻转，背面显示答案；底部两个圆形按钮（记住了/没记住）
- 列表页：错题本/收藏夹，卡片列表，支持删除

## 动效
- 卡片翻转：CSS 3D transform
- 选项错误震动：轻微 shake 动画
- 按钮交互：`active:scale-95` 压感缩放

## 保留与迁移（面向 v0.0.2）
- 将上述色板、尺寸、阴影等抽取为 `tokens`（CSS 变量或 Tailwind theme 扩展）
- 将翻转/震动动画迁移到独立样式文件，组件内按类名复用
- 在 v0.0.2 保持视觉一致性为第一原则
